# DiyaSDK

Base tools for Diya kernel Development

Dependencies on Linux system (Ubuntu focal)

```sh
sudo dpkg --add-architecture i386
sudo apt update
sudo apt install \
    libsdl2-2.0-0 \
    libsdl2-2.0-0:i386 \
    libsdl2-image-2.0-0 \
    libsdl2-image-2.0-0:i386 \
    libsdl2-ttf-2.0-0 \
    libsdl2-ttf-2.0-0:i386
```
